## <text,audio> pair generation

The proposed <text,audio> pairs can be generated by the following steps:

1. Download seed audios from HuggingFace.

2. Edit `config.yaml` to configure the pair generation, such as,

    ```yaml
    save_name: audiorelgen_train.json
    audio_length: 10
    seed_audio_path: seed_audios
    ...
    ```

3. run `main.py` to generate the <text,audio> pairs.

    ```python
    #step1: edit config.yaml
    #step2: generate the pairs
    python main.py
    ```
